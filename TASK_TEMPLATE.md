# Task Template

Use this template before starting a task to keep context compact and execution focused.

## Goal
- What is the single outcome we want?

## Scope
- What is in scope?
- What is out of scope?

## Success Criteria
- What must be true for this task to be done?

## Constraints
- Performance, reliability, compatibility, or deployment limits.

## Notes
- Any important edge cases, files, or risks.
